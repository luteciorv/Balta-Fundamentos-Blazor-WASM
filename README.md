# Balta - Curso de Fundamentos do Blazor WebAssembly

### Descrição
        * Projeto feito com o .NET 7
        * Visão geral do projeto
        * Banco de dados em memória
        * Criação do modelo simples Categoria
        * CRUD de Categorias em uma única página
